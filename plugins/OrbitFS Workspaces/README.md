# OrbitFS Workspaces Addon

This folder is the installable marker and browser bundle for Workspace Mode.

Safety workflow:
1. Open Config > Addons.
2. Detach OrbitFS Workspaces.
3. Move this whole folder out of the plugins directory.

To reinstall:
1. Put this folder back in the plugins directory.
2. Open Config > Addons.
3. Attach OrbitFS Workspaces.

Detaching never deletes workspace files or database records.
